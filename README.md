## 中文

# GoldieRillChat

一个基于 Electron + React 的现代化聊天和图片编辑应用程序。这个应用程序特别适合需要结合聊天和图片处理功能的用户，例如设计师、内容创作者或团队协作场景。它提供了一个集成的工作环境，让用户可以在同一个应用程序中完成多种任务。

## 技术栈

### 前端

- React
- TailwindCSS
- DaisyUI (UI组件库)
- ReactMarkdown (Markdown渲染)

### 桌面应用

- Electron
- Node.js

### 开发工具

- Vite (构建工具)

## English

# GoldieRillChat

A modern chat and image editing application based on Electron + React.This application is particularly suitable for users who need both chat and image processing capabilities, such as designers, content creators, or team collaboration scenarios. It provides an integrated workspace, allowing users to complete multiple tasks within a single application.

## Technology Stack

### Frontend

- React
- TailwindCSS
- DaisyUI (UI component library)
- ReactMarkdown (Markdown rendering)

### Desktop Application

- Electron
- Node.js

### Development Tools

- Vite (Build tool)

## Key Features

### 1. Chat System

- Support for multiple chat sessions
- Session management:
    - Create new sessions
    - Rename sessions
    - Delete sessions
    - Drag and reorder sessions
- Messaging features:
    - Support for text messages
    - Support for Markdown formatting
    - Code block syntax highlighting
    - Edit messages
    - Delete messages
    - Move messages up or down
    - Collapse/expand long messages
    - Copy message content
    - Display message timestamps

### 2. File Handling

- Multimedia file support:
    - Image preview (supports jpg, jpeg, png, gif, webp)
    - Video playback (supports mp4)
    - Display other file types
- File operations:
    - Drag-and-drop upload
    - Clipboard paste
    - Rename files
    - Open file location
    - Delete files (move to recycle bin)

### 3. Image Editor

- Canvas functionality:
    - Adjustable canvas size and aspect ratio (supports multiple preset resolutions)
    - Custom canvas dimensions
- Image editing:
    - Import images
    - Zoom in/out
    - Rotate
    - Drag to adjust position
    - Flip horizontally/vertically
- Editing results:
    - Export to chat
    - Copy to clipboard

### 4. User Interface

- Responsive design
- Collapsible sidebar
- Multi-theme support (18 preset themes)
- Custom scrollbars
- Dark mode support

### 5. Data Management

- Local storage:
    - Auto-save messages
    - File management system
    - Persistent session history
- File system integration:
    - Auto-scan folders
    - File synchronization
    - Recycle bin functionality

### 6. Additional Features

- Keyboard shortcuts support
- Drag-and-drop file operations
- Clipboard integration
- Custom storage path
- Session state retention

## Highlighted Features

- Seamless integration of chat and image editing
- Modern user interface
- Highly customizable theme system
- Comprehensive file management features
- Intelligent message handling system
- Professional image editing tools

## Installation and Usage

1.  Choose a storage location
2.  Create or select a session
3.  Start chatting or editing images
4.  Switch themes and editing modes anytime
